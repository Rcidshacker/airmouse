# AI Air Mouse

Gesture-controlled virtual mouse for Windows. Uses your webcam and MediaPipe hand tracking to control the cursor without touching any hardware.

## Requirements

- Python 3.11+
- Windows 10 or 11
- Webcam with DirectShow driver
- Administrator access (recommended)

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
